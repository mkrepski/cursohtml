Você deve estar pensando:

        "Existem milhares de cursos HTML na Internet, o que este curso têm de especial?"

Segue uma lista:

  1. Este curso é focado nos conceitos da versão 5 do HTML. Muitos cursos ainda usam conceitos ultrapassados, de HTML 4.01 (de 1999) e XHTML (de 2000).
  2. Este curso faz parte da disciplina Desenvolvimento Web, que leciono para alunos de graduação.
  3. O curso ensina como hospedar suas páginas gratuitamente na Internet usando o GitHub Pages.
  4. É 100% gratuito!
  5. Este curso foi projetado para iniciantes e receberá atualizações a partir do seu feedback, ou seja, a partir da sua avaliação positiva e construtiva, este curso crescerá, incorporando conteúdos que te ajudarão a conhecer mais a linguagem e as tecnologias associadas.
  6. Não perca tempo, inscreva-se no curso e comece a aprender HTML5 hoje mesmo!
  7. E compartilhe com seus amigos!

What you’ll learn

  • Compreender os conceitos da linguagem de marcação HTML5
  • Compreender como organizar seu conteúdo HTML5 usando corretamente as estruturas semânticas da linguagem
  • Compreender a importância da boa estruturação de páginas HTML 5 para auxiliar na indexação por parte das ferramentas de busca como Google e Bing
  • Aprender como hospedar suas páginas gratuitamente usando o GitHub Pages
  • Are there any course requirements or prerequisites?
  • O estudante deve estar familiarizado com os procedimentos de instalação de programas em seu computador.

Who this course is for:

- Qualquer um que deseja conhecer os fundamentos do HTML

- Alunos que desejam aprender diretamente na versão mais atual HTML a versão HTML5
